# ASP.NET Core Web API Assignment

A clean and structured **ASP.NET Core Web API** project built with **.NET 8**, demonstrating RESTful API design using multiple controllers and endpoints.

This project was developed as part of an academic assignment to showcase backend fundamentals, controller-based architecture, and API endpoint implementation.
